# web-osint
A simple ruby tool to get info about a website(url)

[![forthebadge](https://forthebadge.com/images/badges/made-with-ruby.svg)](https://forthebadge.com)

# Installation
```
git clone https://github.com/krishpranav/web-osint
cd web-osint
gem install bundler
bundle install
./webosint
```

# Usage
```
./webosint google.com
```
